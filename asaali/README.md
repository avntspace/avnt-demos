# Asaali Bundling Demo

Static AVNT sales demo showing how bundle logic can sit on top of Asaali's existing product page design without changing the core storefront language.

## Stories included

1. `Cannacamo Anorak Jacket`
   - bundle pairing: `Cannacamo Cargo Tech Pants`
2. `Pantone Cartel Hoodie`
   - bundle pairing: `Pantone Cartel Sweatpants`
   - bundle pairing: `Deadhomie$ In Loving Memory Tee`

## What this demonstrates

- Asaali-style PDP layout on desktop and mobile
- bundle attach module placed where Asaali already uses `Add These To The Fit`
- simple cart drawer that keeps the existing Asaali visual language
- story switching for two launch-ready bundle examples
